# Nightreign Companion

A browser-based companion app for Elden Ring Nightreign.

Features:
- Expedition timer with phase tracking (Day 1 / Night 1 / Day 2 / Night 2)
- Real-time search across all reference sections
- Night Lords reference with boss weaknesses and tips
- Relic Effects reference (corrected values from community data)
- Boss Relics (Night of… series)
- Generic Weapon Passives (all tiers, stacking rules)
- Dormant Powers reference
- Legendary Weapon Passives with offhand/sheathed availability info
- Relic Categories rolling rules (20 categories)
- Invasion / Raid Event Buffs reference
